Portifólio

Projeto 4Health

É um aplicativo mobile desenvolvido em Python, utilizando Flet para a interface e SQLite para o armazenamento de dados local. Projetado para facilitar o gerenciamento da saúde do usuário, ele funciona como uma agenda inteligente para consultas, exames e medicações, além de centralizar informações médicas importantes. Como os dados são armazenados exclusivamente no dispositivo, sem necessidade de servidores externos, o app garante maior privacidade e segurança, eliminando riscos de vazamento ou roubo de informações.
